# Converter
This project is a backend-driven unit conversion tool designed to convert between different measurement units, including length, weight, and temperature. Each type of conversion is handled on a separate page to ensure clear structure and easy error detection. The frontend is kept minimal, using basic HTML with an imported stylesheet, while the core functionality focuses on robust backend logic.

Features:
- [x] Converts between millimeters, centimeters, meters, kilometers, inches, feet, yards, and miles (Length).
- [x] Converts between milligrams, grams, kilograms, ounces, and pounds (Weight).
- [x] Converts between Celsius, Fahrenheit, and Kelvin (Temperature).
- [x] Error handling for invalid inputs.
- [x] Modular and documented code, ensuring maintainability and future expandability.
- [x] Local execution, with plans for online deployment experiments.

This project serves as both a learning exercise and a portfolio piece, demonstrating backend logic, structured code, and practical problem-solving in unit conversions.
